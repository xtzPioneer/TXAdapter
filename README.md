## 介绍

* iOS轻量级屏幕以及字体适配，支持iPhoneX、XS、XSMax、XR等机型适配。
* 使用方便、操作简单、集成快捷等优点

## 要求
* 需要在iOS8以上使用
## 安装

TXAdapter is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TXAdapter'
```
